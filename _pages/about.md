## Services

### Software Architecture

- Modular design, encapsulation, aggregation by cluster, affinity analysis, stateless/stateful components, delegation from clients to servers, intellectual property.
- System modeling techniques, context diagrams, data flow diagrams, use case diagrams, process flow charts, activity diagrams, interaction diagrams, sequence diagrams, and state charts.
- Design patterns, MVC, peer-to-peer/hierarchical structures, fork and chain, transactional processing, serivce-oriented design, service contracts.

<br>


### Application Architecture

- Bespoke and third-party platform/user/infrastructure applications, point-to-point, hub and spoke integration, ERP/CRM, screen scrapers, ETL, application consolidation.
- Authentication, authorisation, identification, access, three-factor authentication, enterprise security controls, and misuse policy.
- Remote database access, component distribution middleware, database and transaction middleware, distributed transaction management, and transaction processing.

<br>

### Data Architecture

- Stored data using data models, federated transaction across a distributed databases, storing state outside of databases, database management systems. 
- Data in motion, regular expression, message formats, data flow definitions, canonical data models, and measuring CIA across levels.
- Integrity of data, information domain, data dissemination dimensions, checksum, de/normalisation, data warehouses, security policies, encryption, and digital signature.
